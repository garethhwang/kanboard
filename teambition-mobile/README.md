Teambition Mobile Web App(TMWA)
======================================

## global dependency

```
npm i -g gulp
npm i -g ionic
```
## Compile

```
gulp wechat
gulp ding
gulp qqgroup
```
## Build

```
gulp wechat.release
gulp ding.release
gulp qqgroup.release
```

## deployment
### DingDing in ci
```
npm run locald
```
### DingDing in beta
```
npm run betad
```
## startup
```
ionic serve
```
